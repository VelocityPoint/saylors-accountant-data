# Saylor's Accountant — public data mirror

Read-only mirror of the data tree behind **[saylorsaccountant.com](https://saylorsaccountant.com)** — every weekly Strategy Inc tranche, every SEC source filing, every CSV the live site reads. Auto-synced from a private repo on every change.

## What's here

| Path | What it is |
|---|---|
| `tranches.csv` | Every weekly Strategy Inc 8-K BTC purchase tranche (2020 → present) |
| `strategy-disclosures.csv` | Quarterly cumulative-BTC numbers from 10-K / 10-Q cover pages |
| `cashflow.csv` | Annual cashflow modeling — the USD Reserve buffer math |
| `preferred-ipos.csv` | STRF / STRC / STRK / STRD / STRE IPO terms and proceeds |
| `strc-rate-history.csv` | STRC monthly variable-rate dividend history |
| `filings/8-K/` | Source 8-K filings (PDF + iXBRL HTM) referenced by every row of `tranches.csv` |
| `scripts/` | Parsing / extraction scripts that produced the CSVs from upstream sources |

## Why this exists

The site's promise is *"every number links to its SEC filing."* This mirror extends that to *"every change to the underlying data is in this public repo."* Clone it, fork it, run your own audit harness — the audit trail is `git log`.

## What's not mirrored

- `raw/` — intermediate scraping outputs (strategy.com HTML / JSON snapshots). Format-unstable; not useful for auditing.
- `analysis/` — internal editorial drafts and notes. Not part of the data product.
- The site code itself — that's a private repo.

## Provenance

This mirror is generated by a one-way GitHub Action on every change merged to the upstream private repository. Each commit message in this repo's `git log` corresponds to the upstream commit that triggered the sync — readable as the audit trail of who-changed-what-when.

This repo does **not** accept pull requests. Changes flow one-way from the upstream canonical source. To report a data error, contact **dave.lawler@velocity-point.com**.
